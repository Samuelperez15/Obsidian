## 1.1. BASES DE DATOS: DEFINICIÓN Y VENTAJAS

### ¿QUÉ ES UNA BASE DE DATOS?

Las Bases de Datos (BBDD) son estructuras en las que se almacena un conjunto de datos pertenecientes a un mismo contexto para su posterior recuperación / consulta, análisis o transmisión.  
  
Otra definición menos académica es:  
Una BBDD es una herramienta que recopila datos, los organiza y los relaciona para que se pueda hacer una rápida búsqueda y recuperar con ayuda de un ordenador.  
  
Piensa en una biblioteca como en una BBDD de libros almacenados en lugares concretos en las estanterías, donde hay un archivo con fichas que te permiten encontrar la ubicación exacta del libro buscando por título, autor, categoría, etc.; una BBDD también guarda la información de manera estructurada y nos permite hacer consultas para acceder a esa información y poder trabajar con ella.  
  
En la actualidad, resulta esencial recoger la información en Bases de Datos.

**Tipos de modelos de bases de datos**

La información que contiene una Base de Datos (BBDD) puede ser estructurada de manera distinta, lo que afecta a las operaciones que se realizarán sobre la base de datos (lectura, modificación de datos, etc.). En función de cómo se estructuren los datos de la BBDD, surgen varios modelos.  
  
El modelo relacional es el más utilizado actualmente y da lugar a las Bases de Datos Relacionales. Su sencilla estructura consiste en un esquema de tablas que contienen filas y columnas. Virtualmente, todos los sistemas de Bases de Datos Relacionales utilizan el lenguaje SQL para consultar y mantener la Base de Datos.  
  
En el modelo jerárquico, los datos se organizan en una estructura ramificada, parecida a un árbol. Usa relaciones Padre/ Hijo: un padre puede tener muchos hijos, pero cada hijo solo un padre.  
  
El modelo de red es una Base de Datos compuesta por una colección de registros que están conectados entre sí mediante enlaces en una red.  
  
El multidimensional se representa en forma de cubos de datos, siendo cada una de las caras del cubo la dimensión disponible para los usuarios.  
  
El modelo de datos deductivos es un sistema que permite hacer deducciones a partir de reglas y hechos allí almacenados.  
  
En el modelo orientado a objetos tanto los datos como sus relaciones están contenidos en una única estructura conocida como objeto.  
  
Las bases de datos transaccionales están vinculadas a bases de datos relacionales, y permiten el envío y recepción de datos a grandes velocidades.  
  
Los gestores de BBDD o _Data Base Managenent System (DBMS)_ son sistemas que permiten la creación, gestión y administración de bases de datos.  
  
El modelo documental permite el proceso de examinar los datos para elaborar su índice. Esto nos sirve para realizar búsquedas más potentes y almacenar grandes volúmenes de información.  
  
El modelo orientado a grafos representa la información como nodos o piezas de datos. La relación entre ellos se representa mediante aristas.
