
Es un lenguaje de diseño para lenguajes estructurados:

Cascade Style Sheets.

Describe como deben mostrarse los elementos de HTML

Formas de añadirlo:

·Archivo independiente (la mejor) --> `<link rel="stylesheet" href="style.css"/>`
·Dentro de la etiqueta `<head> con la etiqueta style <head> <style type=text/css">`
·En línea dentro de la etiqueta de un elemento `<h1 style="color:green;"`

El color puesto en el código tiene prioridad respecto al archivo de css pero es una práctica a evitar.

Existen distintos tipos de selectores:
·De tipo: Selecciona todos los elementos del tipo dentro de un documento
·De clase: Selecciona varios elementos
·De ID: Selecciona un elemento unico
