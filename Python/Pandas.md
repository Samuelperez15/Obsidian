
Pandas es un módulo de Python de código abierto para la manipulación y análisis de datos. Proporciona estructuras de datos flexibles y eficientes, como los DataFrame, que se utilizan para trabajar con datos tabulares (similares a una hoja de cálculo) y series temporales. Es una herramienta fundamental para la ciencia de datos y se usa para cargar, limpiar, transformar y analizar datos de diversas fuentes.

`import pandas as pd` --> Esto lo abrevia a pd para cada vez que queramos llamar al módulo

# Dataframe

### Un dataframe es una estructura de datos bidimensional etiquetada que permite almacenar y manipular datos en forma de tabla, similar a una hoja de cálculo o una tabla SQL. Cada columna del DataFrame puede contener datos de diferentes tipos (por ejemplo, enteros, cadenas de texto, números decimales)

## XML

XML es un lenguaje de marcado que define reglas para codificar documentos en un formato que es tanto legible para humanos como para máquinas. Utiliza etiquetas para diferenciar entre los atributos de los datos y los datos reales.

## JSON

JSON es un formato de intercambio de datos ligero y de fácil lectura para humanos y máquinas. Utiliza una estructura basada en pares clave-valor y es independiente de cualquier lenguaje de programación.
