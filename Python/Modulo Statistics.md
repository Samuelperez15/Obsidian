El módulo statistics de Python es una biblioteca integrada para realizar cálculos estadísticos básicos sobre datos numéricos, como la media, mediana, moda, varianza y desviación estándar. Está diseñado para ser una herramienta de nivel de calculadora científica, útil para análisis de datos rápidos y no para sustituir a bibliotecas más complejas como NumPy o SciPy.
